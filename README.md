# Quiz App

## Description
This is a simple interactive JavaScript quiz application built using HTML, CSS, and JavaScript. The app presents multiple-choice questions, tracks the user’s answers, and calculates a final score at the end.

## Features
- Multiple-choice questions
- Next and previous navigation
- Real-time feedback (correct/wrong)
- Score tracking
- Final result display
- Input validation (prevents skipping questions)

## How to Use
1. Open the project in a browser using Live Server or by opening `index.html`.
2. Select an answer for each question.
3. Navigate using the Next and Previous buttons.
4. Click Submit to see your final score.

## Technologies Used
- HTML
- CSS
- JavaScript
